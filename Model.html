In plain words: how do I run one myself?

Closed vs open: "Closed" means the model stays on the company's servers — you use it online and can't host it. "Open" means you can download the model file and run it on your own machine.

Ollama is the simplest tool: install once, then ollama run qwen3.6:27b. Others: LM Studio (friendly app), vLLM (serve to a team), Hugging Face (where the files live).

The catch is hardware: bigger models need more GPU memory (VRAM). A laptop handles small models; a 24GB GPU handles strong ones; the largest open models need a server. Closed models can never run locally.
